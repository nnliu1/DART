## download DART encoder
```
huggingface-cli download anonymous-dart-2026/dart_encoder \
        --local-dir model/best_model
```
## or with git clone
#### Make sure git-xet is installed (https://hf.co/docs/hub/git-xet)
```
winget install git-xet
```
```
git clone https://huggingface.co/anonymous-dart-2026/dart_encoder
```
