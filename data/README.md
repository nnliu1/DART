## download training dateset
```
huggingface-cli download anonymous-dart-2026/training_dataset \
        --repo-type dataset \
        --local-dir data/train
```

## or with git clone
#### Make sure git-xet is installed (https://hf.co/docs/hub/git-xet)
```
winget install git-xet
```
```
git clone https://huggingface.co/datasets/anonymous-dart-2026/training_dataset
```
